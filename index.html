LINE_BOT

使用技術

  • LINE Messaging API
  • php
  • Apache2

詳細

LINE Messaging APIを利用しphpでコードを書いています。,
また、VPSでUbuntuのサーバーを立ててApache2をインストールし、その環境にデプロイしています。

文字数カウントや乱数生成、YouTubeの急上昇ランキング上位の動画の取得などのアクションができるLINE_BOTです。
普段から使い慣れているLINEですべてが完結したら便利だなと思って制作しました。

行うアクションごとにファイルを分割して一つのファイルが大きくなりすぎないように努力しました。
また、入力に対して柔軟に対応できるように条件分岐を工夫しました。

hatena_app

使用技術

  • Swift
  • ChatGPT

詳細

Code.Gate?というプログラミングのサークル内で4日間のハッカソンを行い作成しました。

このアプリは自分の送りたい内容を柔らかい文章にしてくれたり、取引先などに向けて硬い文章にしてくれたりします。
そしてXcodeを用いてSwiftでmacOS用のアプリとして作成しました。

自分はテキストをGPTに送信したり、GPTから受け取った情報を表示したり、アプリのレイアウトなどを担当しました。
Swiftを触るのは初めてだったので、最初はわからないことだらけで難しかったですが、動画などを見て学習して頑張りました。

さぽちむホームページ

使用技術

  • STUDIO

詳細

さぽちむの紹介だったり、活動の記録を残す場所としてホームページとして作成しました。

大学1回生の冬頃から作成を始め、大きなデザイン変更を2回ほど重ねたものが今のホームページになっています。
STUDIOというノーコードのツールを使用しており、STUDIOのアップデートと共にHPも進化していきました。

基本的には自分ひとりで作成しており、特にこだわった部分としてはCMSを利用して定期的な活動報告を簡単に更新できるように工夫したところです。
難しかった部分はレスポンシブ対応で、STUDIOのツールのお陰で比較的簡単に対応できましたが、それでも頭を悩まされました。
